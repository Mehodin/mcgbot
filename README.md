## Building

Python 3.5 or above required.

Install dependencies with `pip`:
```
pip install discord.py python-dotenv tinydb
```

Create a file named `.env` in the project root:
```
DISCORDTOKEN=yourdiscordtoken
PRODUCTION=false
```
